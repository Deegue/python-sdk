# Python-SDK

LeanCloud Python SDK (under development)

## Install

```bash
pip install leancloud-sdk
```

or

```
easy_install leancloud-sdk
```

Maybe you need the `sudo` prefix depends on you OS environment.

## Generate API document

```bash
cd docs
make html
```
